# Dashboard Assignment

This is a responsive dashboard project built with React and Tailwind CSS. It features a custom sidebar with an expandable tree view, various data widgets, and a clean, modern design.

## Features

- **Custom Sidebar**:
  - Dual-pane layout with an icon rail and a detail tree.
  - Expandable/Collapsible folders (Accordion style).
  - No scrollbars for a cleaner look.
- **Dashboard Widgets**:
  - **Revenue Section**: Highlighted metrics with percentage changes.
  - **Top Sales & Best Deal**: Custom card designs.
  - **User Stats**: A scrollable horizontal list of user performance.
- **Charts**: Sparklines and data visualizations using simple SVG components.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS** for styling
- **React Icons** for UI icons

## How to Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  Open https://dashboard-omega-gilt-80.vercel.app/ to view it in the browser.
